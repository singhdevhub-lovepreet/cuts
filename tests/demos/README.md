# Demo recordings

Check-in recordings of the pipeline running end-to-end in the browser, kept here so
progress is easy to review over time. These are **not** automated tests — they are
manual demo captures.

Naming: `YYYY-MM-DD-<what-it-shows>.mp4`

| File | Shows |
|------|-------|
| `2026-07-04-features-5-8-end-to-end.mp4` | Full browser run of features #5–#8 together: upload 6 clips + music → AI brain (Gemini) → 9:16 render, review-UI trim/drop/transition + re-render, and word-by-word captions. |
| `2026-07-04-features-5-8-playback.mp4` | Full-frame playback of the two finished renders: AI-edited scenic montage (reframe + transitions) and the captioned vlog cut. |

`.mp4` is git-ignored repo-wide; this directory is explicitly un-ignored (see `.gitignore`).
